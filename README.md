## 環境構築

```bash
npm install -g cordova
git clone https://github.com/Alesion30/cordovact.git
cd cordovact
npm install
```

webアプリ（dev環境）
```bash
npm run start
```

iosアプリ
```bash
npm run build
cordova platform add ios
cordova run ios
```
